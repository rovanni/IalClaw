Escolha a tela certa para cada tipo de uso.

O IalClaw possui uma experiência simples, pensada para conversa direta, e uma experiência avançada, voltada para acompanhamento técnico, traces e observabilidade.

Modo simples

Converse sem complicação. Peça o que você precisa e o IalClaw responde de forma direta. Essa tela foi pensada para quem só quer conversar com o agente, pedir uma tarefa ou tirar uma dúvida, sem precisar acompanhar grafo, traces ou logs internos.

  • Bom para começar: peça algo simples, como criar um jogo, corrigir um texto ou explicar um erro.
  • Recomendado para usuários leigos e uso cotidiano.
  • Também inclui histórico de conversas e opção de começar um novo chat.

Modo avançado

Use esta tela quando quiser acompanhar o raciocínio do agente, eventos de debug, histórico técnico, replay de traces e o grafo cognitivo em tempo real.

  • Indicado para desenvolvimento, suporte e troubleshooting.
  • Mostra telemetria, repair loop e eventos internos do executor.
  • Ideal para investigar falhas ou validar o comportamento do sistema.
1

Abra a tela adequada

Entre em /simple para uma conversa limpa ou em /advanced para o painel técnico completo.

2

Escreva um pedido claro

Exemplos: “Crie uma landing page para meu produto”, “Explique por que meu projeto falhou” ou “Crie um jogo da cobrinha em HTML”.

3

Use o histórico quando quiser continuar

Tanto no modo simples quanto no avançado você pode abrir conversas salvas e continuar o contexto de uma sessão anterior.

4

Troque para o avançado quando precisar investigar

Se aparecer um comportamento estranho ou você quiser entender o fluxo interno do agente, mude para a tela avançada e acompanhe os traces e eventos.
