Ajuda do dashboard

Escolha a tela certa para cada tipo de uso.

O IalClaw agora tem duas experiências: uma simples, para quem só quer conversar, e outra avançada, para quem quer acompanhar o funcionamento interno do agente.

Modo simples Modo avançado Ajuda
Para uso cotidiano

Modo simples

Ideal para usuários leigos, demonstrações e conversas rápidas. A interface foca só na troca de mensagens e esconde logs, grafo e telemetria.

  • Use para pedir tarefas, ideias, explicações e textos.
  • Recomendado para quem não precisa entender o funcionamento interno.
  • Melhor escolha para apresentações mais limpas.
Para operação técnica

Modo avançado

Ideal para desenvolvimento, suporte e debugging. Exibe grafo cognitivo, traces, replay e logs de eventos do agente em tempo real.

  • Use para investigar falhas e acompanhar decisões do agente.
  • Mostra métricas, telemetria e fluxo interno.
  • Recomendado para times técnicos e troubleshooting.
Como usar
1

Abra a tela adequada

Entre em /simple para uma conversa limpa ou em /advanced para o painel completo.

2

Escreva um pedido claro

Exemplos: “Crie um jogo da cobrinha em HTML”, “Explique esse erro”, “Faça uma página inicial simples para meu produto”.

3

Acompanhe o retorno

No modo simples você vê só a resposta. No modo avançado também acompanha traces, debug e o grafo cognitivo.

4

Troque de tela quando precisar

Você pode começar no modo simples e migrar para o avançado quando precisar investigar algo técnico.
