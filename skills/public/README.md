# Skills Públicas

Coloque aqui skills baixadas de fontes externas.

Fluxo recomendado:

1. Instale ou copie a skill para esta pasta.
2. Rode a auditoria com `skill-auditor` antes de confiar nela.
3. Se aprovada, mantenha a skill aqui.
4. Se for suspeita, mova para `skills/quarantine/`.

Exemplo de organização:

- `skills/public/minha-skill/SKILL.md`
- `skills/public/minha-skill/scripts/...`
